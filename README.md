# SNMPMonitor
Web App that presents information from devices using SNMP Protocol.

###Author: Tiago Ricardo 
###https://www.linkedin.com/in/tiagodricardo

##Instalation and usage

- Install nodejs on or OS;
- Install node_modules: npm install;
- Update on the "devices.json" with the devices;
- And on "index.js" change the OIDs;
- Run: node index.js;
- Open browser on port 8080.
